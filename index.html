MediaElement.js

HTML5 <audio> or <video> player with Flash and Silverlight shims that mimics the HTML5 MediaElement API, enabling a consistent UI in all browsers.

View on GitHub Download .zip Download .tar.gz

Welcome to MediaElement.js.

MediaElement.js has renderers to play the following media formats

  • MP4/MP3 - native <video> and <audio> tags or fallbacks
  • HLS - native, flash, or JavaScript
  • RTMP - Flash
  • YouTube - Common API wrapper
